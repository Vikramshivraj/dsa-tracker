# 🚀 DSA Tracker

A full-stack MERN application to practice DSA problems, track coding progress, compete on real-time leaderboards, and analyze performance with interactive dashboards.

# 📌 Features

✅ JWT Authentication  
✅ Protected Routes  
✅ Real-time Leaderboard using Socket.io  
✅ Online Users Tracking  
✅ Dashboard Analytics  
✅ Problem Solving System  
✅ Difficulty-wise Tracking  
✅ Admin Panel for Problem Management  
✅ Responsive UI with Tailwind CSS  
✅ MongoDB Atlas Integration  
✅ Full Deployment on Vercel + Render  

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- Chart.js
- Socket.io Client

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.io

## Deployment
- Vercel (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
